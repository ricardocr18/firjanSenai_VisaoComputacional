## 👨‍🏫 Como visualizar os projetos

### 💻 Visualização Online
Pressione `.` (ponto) neste repositório para abrir o Editor Web no navegador.

### 📥 Download de uma pasta específica
1. Navegue até a pasta do projeto desejado.
2. No README interno, use o link de **Download Direto (.zip)**.
3. Isso evita baixar todo o repositório desnecessariamente.
